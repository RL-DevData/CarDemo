# 设计说明
相机未锁死,可以设置旋转速度,阻尼等.
无论发生任何事件都可以自由活动.
可以设置车速(轮胎转动速度和地面UV移动速度)

## 遇行人停车事件
远处动态创建斑马线和行人,事件发生后车辆降速.
当降低到300(制作值)后,将远处物体与速度匹配运动(拉近).
当行人通过马路未阻塞后车辆将继续提速到事件前速度.
当物体超出玩家一定区域后删除.

## 自动跟车事件
开启后车辆设置为某个速度.
前方创建目标车辆,并从远处拉近.
当处于一定范围后保持2秒.
然后目标车辆变道后,演示车辆超过目标车辆.
